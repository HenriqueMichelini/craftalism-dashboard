# Craftalism Dashboard (React App)

This directory contains the production dashboard frontend used by the Craftalism ecosystem.

## Authoritative docs

For architecture, setup, and deployment guidance, use the repository root README:

## Local quick start

```bash
npm install
npm run dev
```

Then open the Vite URL shown in your terminal (typically `http://localhost:5173`).
